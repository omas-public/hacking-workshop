# ParrotOSを使ってみよう

[公式サイト](https://www.parrotsec.org/)
ParrotOSとは、セキュリティやペネトレーションテスト、デジタルフォレンジクス、プライバシー保護などを目的としたLinuxベースのオープンソースOSです。多くのハッキング・解析ツールが標準で搭載されており、セキュリティ技術者や研究者、ハッカー向けに設計されています。軽量で使いやすく、匿名性や暗号化機能も充実しています。

## お約束

インストールが始まるのでデスクトップのインストールアイコンはクリックしないこと

## Networkの設定

インターネットにつなぐには、ネットワークの設定が必要です。今回は無線接続を使用します。

- ALT + T で　ターミナルを立ち上げる
- ネットワークの接続状況を確認するために、以下のコマンドを入力します。
  - `nmcli device status | grep wifi` で接続状況を確認します。
  - `nmcli device wifi list` で利用可能なWi-Fiネットワークの一覧を表示します。
  - `sudo nmcli device wifi connect <SSID> password <PASSWORD>` でWi-Fiに接続します。
    - `<SSID>` は接続したいWi-Fiネットワークの名前に置き換えてください。
    - `<PASSWORD>` はWi-Fiネットワークのパスワードに置き換えてください。
  - `nmcli device status | grep wifi` で接続状況を確認します。接続されていれば、SSIDの横に「connected」と表示されます。
  - `ping -c 4 google.com` でインターネット接続を確認します。応答が返ってくれば、接続は成功しています。

## UNIXコマンドの基礎

- pwd （現在のディレクトリを表示する）
- ls （現在のディレクトリの内容を表示する）
- cd ディレクトリ名 （ディレクトリを移動する）
- touch ファイル名 （新しいファイルを作成する）
- echo "テキスト" > ファイル名  （ファイルにテキストを書き込む）
- cat ファイル名  （ファイルの内容を表示する）
- rm ファイル名 （ファイルを削除する）

## 通信の基礎

インターネット通信は、クライアントとサーバーの間で行われます。クライアントはリクエストを送信し、サーバーはレスポンスを返します。HTTPは、WebブラウザとWebサーバーが通信するためのプロトコルです。HTTPリクエストには、GETやPOSTなどのメソッドがあります。HTTPレスポンスには、ステータスコードやヘッダー、ボディなどが含まれます。

## サーバーを立ち上げよう

### sshサーバーを立ち上げる

- `sudo systemctl start ssh.service`
- `sudo systemctl status ssh.service` でSSHサーバーが起動しているか確認します。Active: active (running) と表示されていれば、SSHサーバーは正常に起動しています。

#### お互いのコンピュータに侵入します

- `ip a | grep inet` コマンドを使って自分のIPアドレスを確認して相手に伝えます
- `ssh parrot@相手のIPアドレス` で相手のコンピュータに侵入します(passwordはparrotです)
- `cd ~/Desktop` で相手のコンピュータのデスクトップに侵入します
- `touch hacking{1..20}.txt` でファイルを20個作成します
- 確認が終わったら`rm *.txt`でファイルを消してあげてね

### 通信の中身を覗いてみよう

- `wireshark &` コマンドでスニッファー(盗聴ツール)を立ち上げます
- インターフェースを選んでスニッファーを開始します
- wiresharkのフィルター機能でport番号 5555だけを抽出します
  - `tcp.port == 5555` をフィルターウィンドウに入力します
- ペア間でサーバorクライアントを決め下のコマンドを入力します
  - サーバーの人は `nc -l 5555`
  - クライアントの人は `nc <相手のIPアドレス> 5555`
  - chatが開始されるので何かテキストを入力してください
  - chatができることを確認したらコントロールキーとCキーで終了します
- wiresharkから通信の内容(data)を2文字ずつに分解します
  - ターミナルを新しく開き通信の内容を以下のプログラムを使って文字に直します
  - python3 -c 'print([chr(c) for c in [0x??, 0x??, 0x??...]])
